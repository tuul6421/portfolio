自己紹介

モンゴル市場向けの実用的なWebプロダクトや、実運用を意識した業務システムの開発に取り組んでいます。

React / TypeScript を中心に、データベース設計からUI実装までフルスタックで一貫して対応できます。

  • 認証、管理画面、外部サービス連携を含む本番アプリの開発・公開経験があります。
  • 学習サービスやコミュニティ系サービスなど、人の役に立つプロダクトづくりに関心があります。
